## Política de Garantías

### Garantía de Satisfacción (90 Días Post-Launch)

**Compromiso Neero:**

Si dentro de 90 días después de go-live producción, el sistema NO cumple alguna de
estas garantías:

1. Detección variaciones precio >10% en <1 minuto
2. Reducción 80% tiempo ingreso datos vs manual
3. Dashboards actualizan en <5 minutos
4. Integración SICOM funciona sin errores
5. Agente IA responde en <10 segundos

**Neero remediará sin costo adicional** hasta cumplimiento. Si después de 60 días
remediación no se cumple, **cliente puede solicitar reembolso 50% inversión**.

**Excepciones:** Garantía no cubre errores causados por:
- Modificaciones cliente al código fuente
- Datos SICOM corruptos/inconsistentes (responsabilidad Contecsa)
- Cambios infraestructura cloud sin aprobación Neero
- Fuerza mayor (terremotos, pandemias, guerra)

---

### Garantía de Soporte (6 Meses Incluidos)

**SLA (Service Level Agreement):**

| Tipo Incidente | Tiempo Respuesta | Tiempo Resolución |
|----------------|------------------|-------------------|
| **Crítico** | <2 horas | <24 horas |
| **Alto** | <4 horas | <72 horas |
| **Medio** | <8 horas | <5 días hábiles |
| **Bajo** | <24 horas | <10 días hábiles |

**Definiciones:**
- **Crítico:** Sistema caído, no se puede usar (0 usuarios pueden ingresar)
- **Alto:** Feature core no funciona (ej: alertas no envían, dashboards no cargan)
- **Medio:** Feature secundaria no funciona (ej: export Excel falla)
- **Bajo:** Mejora UX, pregunta configuración

**Canales soporte:**
- Email: soporte@neero.ai (24/7, respuesta según SLA)
- WhatsApp: +57 XXX XXX XXXX (8AM-6PM lun-vie, emergencias 24/7)
- Videollamada: On-demand (agenda con 4h anticipación)

**Después de 6 meses:** Soporte opcional $12M/año (mismo SLA) o cliente auto-soporta

## Modelo de Ownership

### ¿Quién es dueño del código?

**Respuesta:** Contecsa es 100% dueño del código fuente después de pago final.

**Licencia entregada:**
- Código fuente completo (frontend + backend + IA)
- Documentación técnica arquitectura
- Scripts deployment
- **Sin restricciones:** Cliente puede modificar, redistribuir, vender (aunque no recomendamos)

**Neero retiene:**
- Propiedad intelectual de frameworks/metodologías generales
- Derecho a usar caso de éxito Contecsa en marketing (con aprobación)
- Código base genérico (no específico Contecsa) para otros proyectos

**Beneficio cliente:**
- Sin dependencia perpetua de Neero (exit strategy clara)
- Puede contratar developers in-house para mantener/extender
- Puede negociar con otros vendors si Neero no cumple
